\mynonumsect[1.0]{Solving Linear Equations}

\textbf{Goal:} To solve linear equations


\fact{
\myhead{Equality Properties:}
\begin{enumerate}
\item If $x = y$ and \emph{a} is any real number, then
  $x \pm a = y \pm a$.
\item
  If $x = y$ and \emph{a} is any nonzero real number, then $ax = ay$
  and $\frac{x}{a} = \frac{y}{a}$.
\end{enumerate}

\myhead{Solving Linear Equations:}

A \dt{linear equation} is an equation that can be written in the form $ax + b = c$.

To solve a linear equation, follow these steps:
\begin{enumerate}
  \item Remove parentheses using the distributive property.
  \item Combine like terms on each side of the equation.
  \item Add or subtract terms to get the variable on one side and the constant on the other.
  \item Divide both sides by the coefficient of the variable.
\end{enumerate}
}
